# TCP-клиент для отправки имени и получения приветствия

## Описание
Это консольное приложение на C# (.NET), которое реализует TCP-клиент.  
Клиент подключается к серверу, отправляет имя пользователя и получает ответ с приветствием.

В комплекте также идет простой тестовый сервер, который обрабатывает запросы клиента.

## Требования
- .NET 6.0 или выше
- Любая операционная система (Windows, Linux, macOS)

## Запуск (на вашем локальном компьютере)

Скачайте код из репозитория любым способом: через `git clone https://github.com/ваш-логин/Number_1.git` (если установлен Git) или скачайте ZIP-архив со страницы репозитория (кнопка Code → Download ZIP) и распакуйте в любую папку. Затем откройте два терминала (командные строки) на вашем ПК. В первом терминале перейдите в папку `TcpNameServer` и выполните команду `dotnet run` — сервер начнёт слушать порт 3361. Во втором терминале перейдите в папку `TcpNameClient` и также выполните `dotnet run`. Клиент запросит имя, введите его, и вы увидите ответ от сервера. Все команды выполняются на вашем компьютере, GitHub используется только для хранения кода.
