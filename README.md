# TCP-клиент для отправки имени и получения приветствия

## Описание
Это консольное приложение на C# (.NET), которое реализует TCP-клиент.  
Клиент подключается к серверу, отправляет имя пользователя, получает ответ с приветствием и текущим временем, и выводит его на экран.

В комплекте также идет простой тестовый сервер, который обрабатывает запросы клиента.

## Требования
- .NET 6.0 или выше
- Любая операционная система (Windows, Linux, macOS)
